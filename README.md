# BianChat
## 简介
[BianStudio （彼岸工作室）](https://space.bilibili.com/495639032 "B站")基于 .NET Core 6 的模块化聊天平台，其核心使用 BianCore（彼岸核心）来提供高质量的服务。

当前核心架构更新至v2.0.1版本，软件迭代至v1.2版本
